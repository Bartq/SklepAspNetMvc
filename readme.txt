Kurs przeznaczony jest dla wszystkich osób, które chcą nauczyć się od podstaw tworzenia stron internetowych wykorzystując technologię ASP-NET MVC.

ASP.NET MVC – platforma aplikacyjna do budowy aplikacji internetowych opartych na wzorcu Model-View-Controller (MVC) oparta na technologii ASP.NET.

Do najważniejszych cech platformy aplikacyjnej ASP.NET MVC należy:

Wykorzystanie silnika ASP.NET pozwalającego na wykorzystanie wielu komponentów infrastrukturalnych oferowanych przez tę platformę.

Model programistyczny (API) platformy jest mocno oparty na interfejsach, pozwalając na łatwą rozbudowę, dorabianie i testowanie poszczególnych komponentów.

Elastyczny mechanizm mapowania adresów Uniform Resource Locator pozwalający na łatwą budowę aplikacji według wzorca Representational State Transfer (REST), wykorzystujących czytelną dla użytkowników strukturę adresów URL. Mechanizm ten jest  przyjazny silnikom wyszukiwarek internetowych Search Optimization Engine (SEO).
